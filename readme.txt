Euro Rates Shows Euro Exchange Rates recovered daily from European Central Bank feed.

== Description ==

Euro Rates Showsdaily Euro Foreign Exchage Reference Rates. Lite version shows the full list of currencies
provided by ECB feed:

US dollar,Japanese yen,Bulgarian lev,Czech koruna,Danish krone,Pound sterling,Hungarian forint,Lithuanian litas,Latvian lats,
Polish zloty,New Romanian leu,Swedish krona,Swiss franc,Norwegian krone,Croatian kuna,Russian rouble,Turkish lira,Australian dollar,
Brasilian real,Canadian dollar,Ch.yuan renminbi,Hong Kong dollar,Indonesian rupiah,Israeli shekel,Indian rupee,South Korean won,
Mexican peso,Malaysian ringgit,N.Zealand dollar,Philippine peso,Singapore dollar,Thai baht,South African rand.


Updates data directly from European Central Bank.
The reference rates are usually updated by 3 p.m. C.E.T. They are based on a regular daily concertation procedure between central banks across Europe and worldwide, which normally takes place at 2.15 p.m. CET.
All currencies quoted against the euro (base currency)

== Installation ==
<ul>
<li>Download the zip file and extract the contents</li>
<li>Upload the folder ms-eurorates-lite to your WP plugin folder /wp-content/plugins/ directory</li>
<li>Go to Plugins > Installed, and activate the plugin</li>
<li>Go to Appearance > Widgets, and drag the widget to a sidebar</li>
</ul>

Done!
